Environment Setup

This application requires proper configuration of Salesforce Connected App credentials. Make sure the following environment variables are set:

  • SALESFORCE_CLIENT_ID - Your Connected App's Consumer Key
  • SALESFORCE_CLIENT_SECRET - Your Connected App's Consumer Secret
  • SALESFORCE_REDIRECT_URI - The callback URL for OAuth
  • OPENAI_API_KEY - Your OpenAI API Key for ChatGPT
