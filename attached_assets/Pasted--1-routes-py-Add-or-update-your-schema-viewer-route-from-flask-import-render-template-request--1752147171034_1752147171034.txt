✅ 1. routes.py — Add or update your schema viewer route

from flask import render_template, request
from your_project.salesforce import get_salesforce_objects  # adjust this import as needed

@app.route("/schema")
def schema_view():
    # Get the query parameter from the search form
    search_query = request.args.get("q", "").strip().lower()

    # Fetch all Salesforce objects (this is your own function)
    all_objects = get_salesforce_objects()

    # Filter objects based on the search query (case-insensitive label match)
    if search_query:
        filtered_objects = [
            obj for obj in all_objects
            if search_query in obj["label"].lower()
        ]
    else:
        filtered_objects = all_objects

    return render_template(
        "schema_view.html",
        salesforce_objects=filtered_objects,
        search_query=search_query
    )

✅ Note:

    Replace your_project.salesforce with the actual module path you use to get your Salesforce object list.

    obj["label"] assumes your objects are dicts like { "name": "Account", "label": "Account", "custom": False }.

✅ 2. schema_view.html — Add the search form

Replace your current input with this form just above the object list:

<form method="GET" action="{{ url_for('schema_view') }}">
    <div class="mb-3">
        <h5 class="text-white">Salesforce Objects</h5>
        <input 
            type="text" 
            name="q" 
            value="{{ search_query or '' }}" 
            class="form-control bg-dark text-light" 
            placeholder="Search objects..."
        >
</form>
